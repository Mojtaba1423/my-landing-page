# My Landing Page 🌒

A minimalist, high-performance personal landing page. Built with a focus on observation, technical exploration, and clean aesthetics.

## Tech Stack
- **HTML5**: Semantic structure with SEO optimization.
- **SCSS**: Modular and dark-themed styling.
- **JavaScript**: Minimal interactions.
- **GitHub Pages**: Automated deployment.

## Features
- ✅ Fully Responsive
- ✅ SEO & Social Meta Tags
- ✅ Schema.org JSON-LD integration
- ✅ Optimized for WebP images
